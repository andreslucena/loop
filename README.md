A civic-tech prototype built during the [EU Civic Tech Hackathon](https://commission.europa.eu/get-involved/events/eu-civic-tech-hackathon-22-23-june-2026-2026-06-22_en) (22–23 June 2026). Helps citizens discover and respond to public consultation projects relevant to their location and interests.

There are an estimated 5,000–10,000 separate public consultation platforms across Europe — at city, regional, national, and EU levels. Loop brings them together in a single, simple interface.

Built with vanilla JavaScript, Tailwind CSS (CDN), and Google Fonts — no frameworks, no build step, no dependencies. Open `index.html` in a browser to run.
